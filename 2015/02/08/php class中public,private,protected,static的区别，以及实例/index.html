php class中public,private,protected,static的区别，以及实例

2015-02-08

一，public,private,protected的区别
public:权限是最大的，可以内部调用，实例调用等。
protected: 受保护类型，用于本类和继承类调用。
private: 私有类型，只有在本类中使用。
