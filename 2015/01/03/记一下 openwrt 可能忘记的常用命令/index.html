记一下 openwrt 可能忘记的常用命令

2015-01-03

将 /dev/sda1 分区挂载为 swap 交换区：
`mkswap /dev/sda1
swapon /dev/sda1

挂载 ntfs：
`opkg install kmod-fuse ntfs-3g
mkdir sda1
ntfs-3g /dev/sda1 sda1

支持 extroot：
`opkg install block-mount
reboot
