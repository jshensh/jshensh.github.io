linux下SSH传输文件

2014-11-28

通过ssh传输文件
SSH进入对方机器：
`ssh remoteIP

`scp -rp username@remoteIP:/path/filename /path #将远程文件从服务器下载到本地

tar cvzf - /path/ | ssh username@remoteip "cd /some/path/; tar xvzf -" #压缩传输一个目录并解压 ``
