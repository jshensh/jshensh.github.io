解决 wordpress 调用 Google 在线字体加载速度变慢的问题(续)

2014-06-08

sed -i s/fonts\.googleapis\.com/fonts.useso.com/ggrep fonts.googleapis.com -rl ./`

原先的 fonts.233.imjs.work 的规则修改为
`server
{
listen 80;
server_name fonts.233.imjs.work;
index index.html index.htm index.php default.html default.htm default.php;
root /home/wwwroot/fonts.233.imjs.work;

        rewrite ^ /index.php;
        location ~ .*\.(php|php5)?$
                {
                        try_files $uri =404;
                        fastcgi_pass  unix:/tmp/php-cgi.sock;
                        fastcgi_index index.php;
                        include fcgi.conf;
                }

        location ~ .*\.(gif|jpg|jpeg|png|bmp|swf)$
                {
                        expires      30d;
                }

        location ~ .*\.(js|css)?$
                {
                        expires      12h;
                }

        access_log off;
}

然后。。。终于能正常使用了，附截图。。。
